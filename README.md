# Near Miss Analytics Dashboard

## Overview
A React-based dashboard to visualize incident data with interactive filtering.

## Setup Instructions
```bash
npm install
npm run dev
```

## Tech Stack
- React
- Vite
- Tailwind CSS
- Recharts

## Assumptions
I assumed that records with missing dates should be excluded from the Timeline chart but kept in the Total Counts.

## Key Features
- Dynamic KPI Cards
- Interactive Charts (Timeline, Severity, Location)
- Real-time Text Search/Filtering
